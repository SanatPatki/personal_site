Reddit Web Scraper

A command-line tool that lets users interactively surf Reddit posts and track how often a target keyword appears across posts and comments.

Built in Python using the PRAW library for Reddit API access. The tool features a tree-based browsing history, keyword frequency tracking across post titles, bodies, and comments, and a sorted summary of all visited posts at the end of each session. Designed and developed solo as a personal project.

Treasure Castle

A pixel art adventure game about a child sneaking into a gothic castle to hunt for treasure. The project explores meta-horror through a single narrative twist: the moment the character realizes they are trapped in two dimensions. All assets were hand-illustrated in a 16-bit pixel style.

Built with C#, including custom scripts to generate and manage the game's audio files. Developed as a team project.
