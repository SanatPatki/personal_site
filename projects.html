Reddit Web Scraper

2025 Python CLI Tool

A command-line tool for interactively surfing Reddit posts while tracking how often a target keyword appears across titles, bodies, and comments.

Built in Python with the PRAW library. Features a tree-based browsing history, keyword-frequency tracking, and a sorted summary of every visited post at the end of each session. Designed and built solo.

Treasure Castle

2026 C# / Unity Mobile Game Audio Design

A pixel-art adventure about a child sneaking into a gothic castle to hunt for treasure. It explores meta-horror through a single narrative twist: the moment the character realizes they're trapped in two dimensions. Every asset is hand-illustrated in a 16-bit style.

Built with C#, including custom scripts to generate and manage the game's audio. Developed as a team project, where I led audio design.
